# iconify-json-nucleo-credit-cards

> Nucleo **credit-cards** icons as an [Iconify](https://iconify.design/) icon set, prefix `nucleo-credit-cards`.

The licensed SVG data is not shipped. It is generated on your machine from the official Nucleo packages (`nucleo-credit-cards`) during install, gated by your `NUCLEO_LICENSE_KEY`.

```bash
export NUCLEO_LICENSE_KEY=your-license-key
npm install iconify-json-nucleo-credit-cards
```

```ts
import { icons } from 'iconify-json-nucleo-credit-cards';
import { addCollection } from '@iconify/react';

addCollection(icons);
```

CommonJS works too:

```cjs
const icons = require('iconify-json-nucleo-credit-cards');
```

See the [workspace README](https://github.com/JonathanXDR/iconify-json-nucleo#readme) for the full picture.
